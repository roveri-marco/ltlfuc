# LTL_f Unsat Core Extraction

![The LTLfUC approach](https://github.com/user-attachments/assets/5185c567-3a13-42af-b185-af73fd4e4059)

LTLfUC provides algorithms to identify unsatisfiable cores within LTL_f specifications.

This code repo contains the implementation of the techniques described in the following article:

- Marco Roveri, Claudio Di Ciccio, Chiara Di Francescomarino, Chiara Ghidini (2024). Computing Unsatisfiable Cores for LTLf Specifications. In: J. Artif. Intell. Res., 80, 517-558. AI Access Foundation. DOI: [10.1613/jair.1.15313](http://dx.doi.org/10.1613/jair.1.15313)

The implemented solutions leverage the work described in [1] and extend it to handle LTL_f through rewriting into LTL.

[1] Alessandro Cimatti, Marco Roveri, Viktor Schuppan, Stefano Tonetta:
Boolean Abstraction for Temporal Logic Satisfiability. CAV 2007: 532-546
